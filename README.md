# Proyecto_DataMining_NN
Rugby &amp; AI: Minería y Redes Neuronales aplicadas al Top 14 francés


Este proyecto aplica técnicas de Minería de Datos y Redes Neuronales para analizar el desempeño de los equipos y jugadores en la liga Top 14 de Francia. El objetivo es transformar datos crudos de partidos en conocimiento accionable mediante modelos predictivos y descriptivos. Este repositorio es para el proyecto final de Minería de Datos y Redes Neuronales de la maestría en Ciencia de datos de la Universidad Escuela Colombiana de Ingeniería Julio Garavito de Bogotá D.C., Colombia.

This project applies Data Mining and Neural Network techniques to analyze team and player performance in the French Top 14 league. The goal is to transform raw match data into actionable insights using predictive and descriptive models. This repository is used as final project for Data Mining and Neural Networks lecture for Data Science Master Program at Escuela Colombiana de Ingeniería Julio Garavito University in Bogota, Colombia.

Ce projet applique des techniques de Data Mining et de Réseaux de Neurones pour analyser les performances des équipes et des joueurs de la ligue Top 14 en France. L'objectif est de transformer les données brutes des matchs en connaissances exploitables via des modèles prédictifs et descriptifs. Ce dépôt est dédié au projet final de Data Mining et Réseaux de Neurones du Master en Science des Données de l'Université Escuela Colombiana de Ingeniería Julio Garavito à Bogotá D.C., Colombie.

Lenguaje: Python 3.12 

Librerías: Pandas, BeautifulSoup, html, json, Scikit-learn

Datos: LNR Official Stats (Top 14)
